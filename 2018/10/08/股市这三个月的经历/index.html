入市这三月

发表于 2018-10-08 | 更新于 2018-10-17 | 分类于 理财 | | 阅读次数：

上市当天

2018年7月9日9:30，小米集团在香港上市，股票代号1810，纪念小米从2010年成立到2018年上市。发行价定在了17港币，上市当天，小米的各个办公区都在庆祝，当天我很幸运抽到了一个1more耳机，可以算是人生第一次抽奖中奖吧。

开端

其实，在上市的前一个月，我就开始自己研究股票怎么买，开户，入金之类的操作。尤其是港股，在网上查，发现流程还挺复杂的，最好办香港卡，但是招行的香港一卡通需要500万现金才能开户，吓到我了，最后听同事说，他经常用老虎证券，可以用国内的银行卡转账。最终自己捣鼓了一阵，开通了老虎证券的账户，入金遇到了问题。刚开始我不知道入金规则，只存入了几百美金，因为入金太少，过了好多天也没有成功，后来经过给客服打电话，才入金成功。入金后又发现根本就不能交易，提示低于2000美金是不可以建仓交易的。之后就又入了一些，账户中总计大约4400美金，开始了股票之旅。

站在风口上的猪

刚开始交易的时候，几乎啥也不懂，只从一些微信公众号中看过公司的市值，市盈率，建仓，仓位等等几个零星的概念，没有系统的去了解这些东西，甚至在买股票的时候，都没有意识到自己一下子花了接近200%的钱（支持两倍杠杆）的股票，自己还以为没买多少，其实因为美股单位是美金，入金的时候是港币，再买其他的股票发现一直自动撤单，提示资金不足，才想到单位是美金。刚开始自己胡乱交易，买了周黑鸭，京东，虎牙。周黑鸭是从一个公众号主推荐的，说市盈率目前比较低，潜力大，就入了，京东和虎牙纯粹是因为自己经常用或者挺熟悉就买了。刚开始赶上行情好，可以说买啥啥涨，但是我的操作可以说像屎一样，刚赚两三个点，就抛出一部分，然后又买入。。。后期京东一路涨到50，可是呢，我早就抛了，不敢进了，只喝到了一点汤水，从股票赚到了一百多美金。后来就是小米上市了，上市当天，暗盘交易已经破发，到16.6左右，当天收盘16.8，处于破发的状态，之后的几天是一波大涨的行情，我还是那么谨慎，就是看涨了就抛，也不懂啥时候接近顶部，来回折腾，也小赚了一波，加上映客上市，前两天也赚了几千港币。这时候其实已经被暂时的盈利冲昏了头脑，感觉股票赚钱真容易啊，还上啥班呢🤐

糊涂的再次入金

小米上市后一段时间，赚了一些钱，有朋友就说让我代买股票，给我打钱，我也又凑了一些钱，加上老虎证券的环球账户低于2.5万美金，有每5个交易日3次T+0交易（当天买卖）的限制，而标准账户没有这个限制，也没有弄清楚这两种账户到底有什么区别，就知道好像只是有些股票买不了，不能做空之类的，感觉也没啥影响，最后就入金3900美金到了标准账户。

一路下跌的股市和更奇葩的交易

入金之后，两个账户本金8300美金，经过刚开始的一段时间，总共盈利了900美金后，小米，映客开始一路跌，但是呢，傻傻的我坚信跌了就补仓呗，越跌越买，结果最后没现金了，还在跌，到后来，映客赔了很多，两个账户都买了不少，映客一路从5块跌到1.5，刚上市就这样，连跌十几个交易日。后来小米也是，一路破发。当时自己买了好多其他的股票，把资金弄的很分散，最终结局就是跌了补仓也补不了，只能割肉，一路亏损到6300美金，赚的900没了，本金也赔了2000。

开始学习技术指标

后来经过赔钱感觉不能这样下去了，开始自学一些技术指标，MACD，KDJ之类的。每天看历史K线的指标，做出下一步的预测分析和记录，慢慢开始有了恢复，赚了点钱，本金回到了7300左右。但是由于标准账户当时不能做空，所以就只能做一个方向，盈利比较慢。

要命的阿里财报&期权操作

在八月份的一波财报季，买了阿里的看涨期权。阿里发财报的那天晚上，看着表面数据还行。同时，因为也是第一次买期权，没有分寸，直接买了10手，随着刚开盘的拉升，不到5分钟，盈利了300美金，当时的心情是很激动的，因为能在这么短的时间内赚不少钱，内心开始膨胀，加上在止盈的时候没有全部卖出，再加上股价在5分钟后又来了一波跌和涨，让我后悔没有多买一些，内心已经开始混乱了，后来又过了一会儿，阿里股价又跌了下来，我就开始加仓，想着能再来一波，没想到一直跌，我也就一直补仓，最终把环球账户里剩余的现金和杠杆都用来补仓，结局是，没一会儿，已经出现了900美金的亏损，内心开始无法承受，但是已经没有现金补仓，也没有及时止损。后来，更悲剧的事情发生了，阿里一路再跌，期权已经接近废纸，而我，还是没有止损，心存侥幸，明天可能会涨回来吧，结果是第一天亏损了1100美金，第二天更多，又亏损750美金，这样以后，我就发烧了，浑身无力，头痛欲裂，请了一天假，内心遭到了巨大的打击。第三天回弹了250美金，但是真的是杯水车薪，可是我没有在这个时候选择离场，仍然心存侥幸会再涨回来一点，但是因为已经临近行权日期，期权越来越不值钱，最终就是，期权亏损了97%，变成了毛票，卖的钱还不够手续费。最终阿里巴巴的0831call赔了2071美金（不包含手续费）。环球账户的资金跌到了1750美金。

又玩了两把期权

经历了阿里期权严重亏损，过了不到半个月以后，阿里要反弹，又买了两次期权，但是因为这次买的很少也没赚多少钱，两个加起来只赚了两百多美金，使得环球账户过了2000美金线（老虎要求2000美金以上才能进行融资融券，就是加杠杆）。

老实看技术指标

后来又买了点期权，没有赚钱，看到优信（买优信是因为刚开始玩股票的时候亏损了一些，想赚回来）和华米（2018的发布会前几天），买完之后的几天里，他们都没有大的涨幅，后来华米在发布会以后涨了，就赚了几个点出货平仓了。

让人再次贪婪的趣头条

9月18号趣头条上市，因为我之前体验过几次趣头条的App。所以还是关注了一下，上市当天，刚开始交易触发了好多次美股的熔断机制（短时间内波动太大，达到一定程度就会触发停牌，几分钟后恢复交易）当天最高涨到了180%，这足以让很多投资者为之疯狂。但是已经涨了这么多，大家也不敢入，经过第二天和第三天，一路从20跌到9.8，我没忍住，入手了，结果就是一路下跌，因为贪婪，又没有止损，还是一路补，后面跌到6块多此时已经赔了30%，而且看K线还在下跌行情，但是真的是账户没钱了，（此时我的内心在想，一旦割肉就真的是亏损了，已经亏不起了），在此之前，买的优信也没有大的涨幅，反而小幅下跌，甚至盘中还达到了我的止损价格，浪费一次T+0，卖出的钱都用来补仓趣头条，结果还是一路跌。第二天优信从底部开始反弹，几天涨了15个点，再看趣头条，不禁唏嘘不已。

对低于2000美金的误解

当我的环球账户跌到了2000美金以内，我企图通过买期权来快速回本。但是发现提交后自动撤单，刚开始以为是买了趣头条，自动撤单，提示包含借贷超过2000美金，我就把趣头条给全部卖了（此时亏损30%），但是又提示我账户现金不足2000，不能融资融券，导致我以为低于2000不能买期权（但是印象中上次低于2000时是买成功了）。难道还要再次入金吗？？后来我仔细看了一下老虎里面的说明，撤单提示中包含了保证金几个字，发现我的保证金是1.5美金，才发现原来我的港币欠了点钱（因为买了港股），我把它还了之后，就可以正常提交期权订单了！趣头条也不该这么着急平仓！原来是我不懂规则。

标准账户的京东带来的又一波亏损

通过K线图，感觉京东已经超跌要反弹了（只看了日k的KDJ，没有看周线趋势和其他指标），就买了正股（标准账户只能买正股），结果周末就爆出来刘强东在美国被关到监狱里，股价应声下跌，而且是跌跌不休，从30跌到最低22.5，而我还是依然傻傻的做T降低成本。

账户的至暗时刻

此时多重因素叠加，账户里只剩下4000美金了，和刚开始8300相比已经赔了一半，此时已经是10月，十一回到家，给家里人说了这件事，他们不懂股票，问我是不是被骗了，是不是公司让我买的，我说不是，是我自己要买的。给他们解释了好久，最后他们说，就当你还在学校上学没挣钱吧（确实把毕业后挣的钱都投进入了，我大姐此时家里房子装修要花钱，我也没有帮上忙）

环球账户全心做期权

通过10月4号意识到不到2000美金也可以做期权之后，我过开始重新操作了期权，每只期权只买两手到三手，买了6只，分散风险，通过确认日k和周k的趋势才去买，买了哔哩哔哩和苹果的put，其他买了call，果然到了第四天以后，所有的期权都赚了，都看对！四天收获了大约600美金。

赶上了大事情

在其他期权都赚钱后，我买了Yandex的call，发现没有赚钱我就又补仓了，但是恐怖的一天出现了，美股纳斯达克大盘跌了超过4个点，期权腰斩，四天的利润全部回吐，白干，心情很失落。第二天上班整个人都不开心，所幸，当天晚上美股没有再次严重下跌，高开了不少，虽然大盘最终还是跌，但是已经好了不少。

Yandex发现操作问题

没有即使止损，但是两手亏损30%左右，也就不到150美金，但是我想着可以补仓赚回来，又忘了之前的教训，导致了上面的问题，后来在涨上来一些之后，卖出了一半的仓位，降低风险，同时可以在涨的时候回点本

1013的交易

美股崩盘第三天，收盘时账户总资金5000美金，比前几天多了1000美金，环球账户因为现在资金不多，全部用来买期权，等回本了，会把标准账户的钱出金（标准账户有许多不好的地方）。

三个月的一些总结

  1. 看对不看涨
  2. 顺势而为，趋势成型再进场
  3. 不要贪婪，达到心里预期就出场
  4. 不带个人情感选股票
  5. 美股期权买卖最好T+0,选择两到三周后的末日期权，如果不能T+0,看15分钟趋势线再进场，15，5，3分钟趋势线都看对
  6. 刚开盘如果下跌，一定要看5分钟线上涨趋势再补仓！如果看错了，趋势发生变化，不要抱着侥幸心理会反弹，果断平仓。
  7. 一定要看大盘
